## New Features

### 1. **Difficulty System**
- Four difficulty levels: Easy, Medium, Hard, Expert
- Each difficulty affects:
  - Time limits
  - Obstacle frequency
  - Node values
  - Complexity

### 2. **Daily Challenges**
- New challenge generated every day
- Random difficulty and objectives
- Earn coins upon completion
- Track completion status

### 3. **Leaderboard**
- Local leaderboard tracking top scores
- Filter by difficulty level
- Shows top 100 entries
- Displays player name, score, level, and difficulty

### 4. **Theme Shop**
- 6 themes available:
  - **Midnight** (default) - Dark blue with vibrant red
  - **Ocean** - Deep blue with cyan
  - **Forest** - Dark green with vibrant green
  - **Sunset** - Purple with orange
  - **Neon** - Dark purple with pink
  - **Golden** - Dark brown with golden yellow
- Purchase themes with earned coins
- Apply themes instantly
- Themes persist across sessions

### 5. **Coins System**
- Earn coins by:
  - Completing levels
  - Finishing daily challenges
  - Achieving high scores
- Spend coins on:
  - Theme unlocks
- Track balance in home screen

### 6. **Enhanced Home Screen**
- Coins display with shop access
- Daily challenge card
- Quick play with difficulty selection
- Statistics overview
- Leaderboard preview
- Achievements preview
- Theme customization button

## Key Game Mechanics
1. **Node Navigation:** Tap nodes in sequence to create paths
2. **Combo System:** Chain successful moves for higher scores
3. **Time Challenges:** Beat levels before time expires
4. **Power-ups:** Collect special nodes for bonuses and extra time
5. **Obstacles:** Avoid red nodes that reduce your score
6. **Difficulty Scaling:** Choose your challenge level

## Settings Features
- Haptic feedback control
- Progress statistics display
- Reset progress option
- Delete account (full reset)
- Achievement tracking

## Onboarding Flow
Five-screen onboarding introducing:
1. Welcome to Luminal Quest
2. Core gameplay mechanics
3. Progress & competition features

Uses `@AppStorage` to track completion state.

## Achievement System
- Complete levels to unlock achievements
- Track total score and playtime
- View progress in home screen
- Multiple achievement categories

## Game Flow
1. **First Launch:** Onboarding experience
2. **Home Screen:** Access all features
3. **Select Difficulty:** Choose challenge level
4. **Play Level:** Complete objectives for rewards
5. **Earn Coins:** Use for theme customization
6. **Daily Challenge:** Fresh challenge every day
7. **Leaderboard:** Compete for top scores
8. **Progression:** Unlock new levels

## Data Persistence
All data stored locally using UserDefaults:
- User progress and scores
- Completed levels
- Unlocked achievements
- Theme preferences
- Purchased themes
- Coin balance
- Daily challenge status
- Leaderboard entries
